# attendance-management-system-appscript

A complete Employee Attendance Management System built using Google Apps Script and Google Sheets.

## Features
- Employee Login
- Admin Login
- Clock In / Clock Out
- Attendance Tracking
- Leave Management
- Holiday Management
- Salary Records
- Dashboard Analytics

## Tech Stack
- HTML
- CSS
- JavaScript
- Google Apps Script
- Google Sheets
- Chart.js
